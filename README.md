## JavaScript Deep Dive Backend - Team 1

### Requirements
- Node
- npm

### Install Dependencies
```bash
npm install
```

### Environment variable
```bash
.env.sample cp .env
```

### Run Application
```bash
npm start
```
OR

```bash
npm run watch
```
